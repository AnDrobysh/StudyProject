Упражнения по модификаторам доступа

Упражнение: Изучение уровней доступа
Создайте два класса ClassA и ClassB в одном пакете.
В ClassA объявите:
поле private int privateField и метод private void privateMethod().
поле protected int protectedField и метод protected void protectedMethod().
поле int defaultField и метод void defaultMethod().
поле public int publicField и метод public void publicMethod().
В ClassB создайте объект класса ClassA и попытайтесь обратиться ко всем полям и методам.
Проанализируйте, какие члены доступны, а какие нет, и объясните почему.

Упражнения по ключевому слову final

Упражнение: Использование final переменных
Создайте класс Circle с полем final double PI = 3.14.
В классе Circle создайте метод calculateArea(double radius), который использует значение PI для расчета площади круга.
Попробуйте изменить значение PI в методе и объясните, почему возникает ошибка компиляции.

Упражнения по ключевому слову static

Упражнение 1: static методы и поля
Создайте класс MathOperations с static полем int counter и методом static void incrementCounter(), который увеличивает значение counter на 1.
В методе main() создайте несколько объектов MathOperations и вызовите incrementCounter() несколько раз.
Объясните, почему значение counter остается общим для всех объектов.

Упражнения по ключевому слову abstract

Упражнение 1: Реализация абстрактных методов
Создайте абстрактный класс Vehicle с абстрактным методом move().
Создайте два подкласса Car и Bike, которые наследуют Vehicle и реализуют метод move().
В методе main() создайте объекты Car и Bike и вызовите метод move() для каждого.
Объясните, почему абстрактный метод должен быть реализован в подклассах.
