Упражнение 1: Работа с List

Задание: Напишите программу, которая принимает на вход список чисел, добавляет в него числа от 1 до 10, удаляет все четные числа и выводит список на экран.

Подсказки:

    Используйте List<Integer>.
    Для удаления элементов можно использовать метод removeIf().

Упражнение 2: Работа с Set

Задание: Напишите программу, которая принимает на вход массив строк, создает множество уникальных строк, а затем проверяет, есть ли во множестве определенная строка (например, "Java").

Подсказки:

    Используйте Set<String>.
    Множество не содержит дубликатов, так что все повторяющиеся строки автоматически удалятся.

Упражнение 3: Работа с Map

Задание: Напишите программу, которая создает отображение (Map), где ключом будет название страны, а значением — ее столица. Заполните Map тремя странами и их столицами, затем выведите на экран все страны и их столицы. Проверьте, есть ли в Map определенная страна (например, "Россия").

Подсказки:

    Используйте Map<String, String>.
